## **4. План улучшения качества**

**Приоритет 1 (критично):** Обработка пропусков в health condition:  
- Провести анализ паттернов пропусков (MCAR/MAR/MNAR)  
- Заполнить 71.29% пропусков значением "Не указано" для сохранения структуры данных  
- Для оставшихся данных рассмотреть multiple imputation с использованием age, bmi, blood_pressure  

**Приоритет 2:** Коррекция выбросов в avg_heart_rate:  
- Проанализировать выбросы в контексте activity_type и intensity  
- Применить winsorization (заменить значения выше 95-го процентиля)  
- Удалить только физиологически невозможные значения (>220 уд/мин в покое)  

**Приоритет 3:** Обработка некорректных значений в daily_steps:  
- Заменить все отрицательные значения (-419 шагов) на 0, так как физически невозможно пройти отрицательное количество шагов  
- Дополнительно рассмотреть замену на медианное значение (8607 шагов) или среднее значение по соответствующей возрастной/половой группе  
- Внедрить автоматическую валидацию данных при сборе для предотвращения записи отрицательных значений в будущем  

**Приоритет 4:** Расширение и обогащение данных:  
- Собрать данные о дополнительных медицинских условиях  
- Включить участников старше 65 лет для полноты возрастного спектра  
- Добавить единицы измерения для всех числовых признаков
