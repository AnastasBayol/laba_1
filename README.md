# **Лабораторная работа №1: Первичное исследование и оценка качества данных**

## **Описание проекта**

Первичный анализ датасета \"Health and fitness dataset\" в рамках дисциплины "Интеллектуальный анализ данных".

**Предметная область:** Данные о фитнес-активности и показателях здоровья, собираемые через носимые устройства. Содержат информацию о физической активности, сердечном ритме, сне, стрессе и других биометрических показателях за 2024 год.

**Цель лабораторной:**

1.  Научиться описывать датасет в терминах задачи интеллектуального анализа данных (кто заказчик, какие вопросы к данным).

2.  Выполнить первичный обзор данных: размер, структура, типы признаков, базовая статистика.

3.  Провести базовый аудит качества данных: пропуски, дубликаты, подозрительные значения, выбросы.

4.  Построить несколько простых, но осмысленных визуализаций для разведочного анализа.

5.  Оформить результаты в виде:

-   воспроизводимого Jupyter Notebook;

-   краткого текстового отчета с выводами и планом улучшения качества данных;

-   аккуратно организованного репозитория в системе контроля версий.

## **Структура проекта**

- `notebooks/01_data_understanding.ipynb\` - основной Jupyter Notebook с анализом

- `report/quality_report.md\` - текстовый отчёт с выводами и планом улучшения

- `data/health_fitness_dataset.csv\` - датасет или скрипт загрузки

## **Инструкции по запуску**

1.  Установите зависимости:  
pip install pandas numpy matplotlib seaborn jupyter
2.  Запустите Jupyter Notebook:  
jupyter notebook  
3.  Откройте файл `notebooks/01_data_understanding.ipynb`
